Elige tu interfaz favorita:

1. 🌐 INTERFAZ WEB (Más moderna y visual)
   → Doble clic en: run_web.bat
   → Se abre en tu navegador
   → Dashboard interactivo con gráficos

2. 🖥️ INTERFAZ DE ESCRITORIO (App nativa)
   → Doble clic en: run_gui.bat
   → Ventana de aplicación tradicional
   → Funciona sin navegador

3. 💻 LÍNEA DE COMANDOS (Para expertos)
   → Abre PowerShell en esta carpeta
   → python src/main.py
